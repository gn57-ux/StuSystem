# reports-export — 需求规格

## 概述

实现班级考试报告页、学生个人学情报告页，支持浏览器打印/PDF 导出，以及成绩数据的 CSV 导出。

## 项目信息

- 项目名: student-performance
- 架构类型: Turborepo monorepo（React + Hono/tRPC + Drizzle + PostgreSQL + Better Auth）

## 需求版本

| 日期 | 版本 | 说明 |
| --- | --- | --- |
| 2026-06-29 | v1 | 初始需求 |

## 用户故事

- 作为教师，我想要生成一份班级考试报告，以便在家长会或教研活动中展示
- 作为班主任，我想要生成学生个人学情报告，以便与学生或家长沟通
- 作为教师，我想要导出原始成绩数据为 CSV，以便在 Excel 中做进一步分析

## 功能需求

1. [F-001] 班级考试报告页：关键指标摘要（平均分、及格率、优秀率）、分数段分布图、科目均分对比图、进退步学生列表、重点关注学生列表、教师评语（可编辑文本）
2. [F-002] 学生个人学情报告页：学生基础信息、最近考试成绩摘要、总分趋势图、科目雷达图、预警摘要、教师备注摘要
3. [F-003] 打印友好布局：通过 `@media print` CSS 隐藏导航栏、按钮，调整字体/颜色适合打印
4. [F-004] PDF 导出入口：「打印/导出 PDF」按钮触发 `window.print()`（浏览器原生打印到 PDF）
5. [F-005] 成绩 CSV 导出：按考试 + 班级 + 科目筛选，导出对应成绩数据（学号、姓名、科目、分数、状态、排名）
6. [F-006] 报告内教师评语：独立的可编辑 `<Textarea>`，内容通过 tRPC 保存到服务端（关联 examId+classId）

## 非功能需求

- 打印样式：报告页打印时隐藏侧边栏、筛选栏、按钮；图表转为静态（Recharts SVG 天然可打印）
- 导出：CSV 导出在前端生成文件（Blob + URL.createObjectURL），不经过服务端文件系统

## 验收标准

- [ ] [AC-001] 班级报告页展示关键指标卡片 + 至少 2 个图表
- [ ] [AC-002] 点击「打印/导出 PDF」触发浏览器打印对话框，预览中侧边栏不可见
- [ ] [AC-003] 学生个人报告页展示基础信息 + 趋势图 + 雷达图
- [ ] [AC-004] CSV 导出：下载文件包含表头行和数据行，列包括学号、姓名、科目、分数
- [ ] [AC-005] 教师评语保存后刷新页面仍保留
